📊 Swiggy Data Analysis | MySQL & Power BI
📌 Project Overview

This project performs end-to-end data analysis on Swiggy food order data using MySQL and Power BI.
The workflow includes data cleaning, data warehouse creation (Star Schema), SQL-based business analysis, and interactive dashboard development.

🗂️ Project Workflow

Imported raw CSV data into MySQL

Performed data cleaning & validation

Removed duplicates using SQL window functions

Designed Star Schema Data Warehouse

Created Fact & Dimension tables

Performed KPI and business analysis using SQL

Connected MySQL with Power BI for dashboards
